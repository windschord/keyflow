# DEC-006: 再生音色にSalamander Grand Piano V3サンプル + Tone.Samplerを採用

## ステータス

承認済（2026-07-07、ユーザー選択）

## コンテキスト

US-013で再生音色の選択機能が要件化された。アプリはApache License 2.0での公開を予定しており、同梱音源はそのライセンスと共存可能（再配布可・商用可）である必要がある。また外部ランタイム依存なし・オフライン動作が製品前提のため、音源はアプリに同梱しなければならない。

## 検討した選択肢

| 選択肢 | ライセンス | 評価 |
|--------|-----------|------|
| A) Salamander Grand Piano V3 | CC-BY 3.0（要クレジット表記） | 採用。Yamaha C5実録・16ベロシティレイヤーの高品質。Tone.js公式サンプルでの使用実績。クレジット1行で配布条件を満たす |
| B) VCSL / VCSL Keys | CC0（表記不要） | 品質はSalamanderに一歩譲る。将来の音色追加候補としては有力 |
| C) FluidR3_GM | MIT | SF2形式でTone.js非対応（要変換）。「全体を丸ごと配布」条件が間引き同梱と相性が悪く不採用 |
| D) シンセのみ（サンプルなし） | - | 「本物のピアノ音」の要望を満たせず不採用 |

## 決定

- **Salamander Grand Piano V3（CC-BY 3.0）を間引いて同梱**し、`Tone.Sampler` で再生する
  - 短3度間隔×単一ベロシティレイヤー（約30ファイル、mp3、合計20MB以下）
  - クレジット「Salamander Grand Piano V3 by Alexander Holm (CC-BY 3.0)」をREADMEとAboutページに表記
- シンセプリセット（エレピ・オルガン・シンセ）を`Tone.PolySynth`で追加し、計4音色から選択可能とする
- Apache 2.0（コード）とCC-BY 3.0（音源アセット）のデュアル構成をREADMEに明記する

## 影響

- アプリ配布サイズが約10-20MB増加する
- サンプルロード待ちのUX（REQ-013-003: ロード完了まで再生開始を待つ）が必要になる
- Aboutページ（US-015）がクレジット表記義務の受け皿となる
