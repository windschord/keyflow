# ユーザビリティ要件

## 概要

SynthesiaやPiano Marvelに匹敵する直感的な操作性と視覚的なわかりやすさを実現する。
ピアノ初心者から上級者まで使いやすいインターフェースを目指す。

## 要件一覧

| ID | 要件概要 | 優先度 |
|----|---------|--------|
| NFR-U-001 | 主要機能へのアクセス（3クリック以内） | 必須 |
| NFR-U-002 | 日本語UI対応 | 必須 |
| NFR-U-003 | キーボードショートカット | 推奨 |
| NFR-U-004 | ダークモード対応 | 推奨 |
| NFR-U-005 | 高DPI（Retina）ディスプレイ対応 | 必須 |
| NFR-U-006 | ウィンドウサイズに応じたレイアウト調整 | 必須 |

## 詳細要件

### NFR-U-001: アクセシビリティ

- **NFR-U-001**: ユーザーがアプリを起動した後、MusicXMLファイルを開いて練習を開始するまでの操作は3クリック以内で完了しなければならない

### NFR-U-002: 言語対応

- **NFR-U-002**: システムのUIは日本語を標準言語として表示しなければならない
- **NFR-U-002b**: システムは将来の多言語対応のためにi18n（国際化）対応の実装をしなければならない

### NFR-U-003: キーボードショートカット

- **NFR-U-003**: システムは以下の主要操作をキーボードショートカットで実行できなければならない
  - スペース: 再生/停止
  - Ctrl+O: ファイルを開く
  - Ctrl+左右矢印: 前後の小節へ移動
  - R/L/B: 右手/左手/両手モード切替

### NFR-U-005: 高解像度対応

- **NFR-U-005**: システムはWindows・macOSの高DPIディスプレイ（Retina含む）でも楽譜や鍵盤が鮮明に表示されなければならない

### NFR-U-006: レスポンシブレイアウト

- **NFR-U-006**: システムは最小ウィンドウサイズ1280×720から最大化まで、楽譜と鍵盤エリアのレイアウトを適切に調整しなければならない
