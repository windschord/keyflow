# US-010: 曲の再生（お手本演奏）

## ユーザーストーリー

**私は** ピアノ練習者として
**〜したい** 読み込んだ曲を再生してお手本として聴きたい
**なぜなら** 練習を始める前に曲の全体像・リズム・テンポ感を耳で把握したいから

## 受入基準（EARS記法）

### 機能要件

- **REQ-010-001**: ユーザーが再生操作（再生ボタンまたはスペースキー）を行った時、システムは読み込み済みの楽曲を現在の再生位置から演奏しなければならない
- **REQ-010-002**: 楽曲が読み込まれていない場合、システムは再生コントロールを無効化し、無効である理由をツールチップで提示しなければならない
- **REQ-010-003**: 再生中にユーザーが一時停止操作（一時停止ボタンまたはスペースキー）を行った時、システムは再生位置を保持したまま演奏を停止しなければならない
- **REQ-010-004**: ユーザーが停止操作を行った時、システムは演奏を停止し、再生位置を先頭（ループ有効時はループ開始小節）に戻さなければならない
- **REQ-010-005**: 再生中、システムは楽譜上のカーソルを再生位置に同期して移動させなければならない
- **REQ-010-006**: 再生中にユーザーがテンポを変更した時、システムは再生速度を即時に反映しなければならない（REQ-006-002と整合）
- **REQ-010-007**: 再生中、システムはMIDI入力の正誤判定（US-004）を一時停止し、再生の停止または一時停止後に判定を再開しなければならない
- **REQ-010-008**: ループ範囲（US-007）が有効な状態で再生位置がループ終了小節の末尾に達した時、システムはループ開始小節から再生を継続しなければならない
- **REQ-010-009**: ユーザーがツールバーの音量スライダーを操作した時、システムは伴奏・メトロノーム・効果音のマスターボリュームを即座に変更し、値をelectron-storeへ永続化しなければならない（スライダー値0はミュートとする。TASK-052、2026-07-05実機フィードバック原因群D対応）

## 画面/UI要件

- 再生/一時停止/停止ボタンをツールバーの目立つ位置に配置する（日本語ラベルまたはアイコン+日本語ツールチップ）
- スペースキーで再生/一時停止をトグルする（NFR-U-003と整合）
- 再生状態（停止中/再生中/一時停止中）が視覚的に判別できること

## データ要件

- 再生状態（stopped / playing / paused）と再生位置は練習セッション内で保持する
- 再生音量はマスターボリュームとしてユーザーが調整可能とし、electron-storeに永続化する（TASK-052で実装昇格。関連: REQ-010-009）。音色の設定は引き続き将来の拡張とし、本ストーリーでは既定値を用いる
