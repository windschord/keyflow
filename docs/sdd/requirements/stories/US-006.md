# US-006: テンポ調整

## ユーザーストーリー

**私は** ピアノ練習者として
**〜したい** 練習するテンポを自由に変えて、ゆっくりから徐々に速くしていきたい
**なぜなら** 難しいパッセージをゆっくりから練習して習得速度を上げたいから

## 受入基準（EARS記法）

### 機能要件

- **REQ-006-001**: システムはMusicXMLに記述されたテンポ記号をデフォルトテンポとして設定しなければならない
- **REQ-006-002**: ユーザーがテンポスライダーを操作した時、システムは即座に再生テンポを変更しなければならない
- **REQ-006-003**: システムは元のテンポの20%〜200%の範囲でテンポ変更をサポートしなければならない
- **REQ-006-004**: テンポを変更しても、システムは音の高さ（ピッチ）を変えずに速度のみを変更しなければならない
- **REQ-006-005**: ユーザーがメトロノームを有効にした時、システムは設定されたテンポでクリック音を鳴らしながら練習できなければならない
- **REQ-006-006**: ユーザーが特定のBPM値を直接入力した時、システムはその値をテンポに設定しなければならない
- **REQ-006-007**: MusicXMLにテンポ変化（ritardando等）が含まれる場合、システムはそれを再現しなければならない

## 画面/UI要件

- テンポスライダーとBPM数値入力フィールドを練習画面の目立つ位置に配置する
- メトロノームON/OFFトグルボタンを隣に配置する
- 元のテンポへのリセットボタンを設ける

## データ要件

- 現在のテンポ設定は練習セッション内で保持する
- 曲ごとの最後に使用したテンポ設定を保存する
